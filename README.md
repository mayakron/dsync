## Features

DSync synchronizes the content of two directories using multiple difference and mirroring modes, and can be effectively used for file-based application deployments.

## Download

All versions of DSync can be downloaded from [here](https://github.com/mayakron/dsync/releases).

## Installation

DSync is portable: just expand the archive to a directory of your choice and run the DSync.exe file.
